Respuesta de la Pregunta#3:

Java y PHP son dos lenguajes de programación muy diferentes y cada uno tiene sus propias ventajas. Java es un lenguaje orientado a objetos que se utiliza a menudo para construir aplicaciones empresariales a gran escala. PHP, por otro lado, es un lenguaje de scripting del lado del servidor que se utiliza comúnmente para el desarrollo web. La elección entre los dos dependería de los requisitos específicos del proyecto.
